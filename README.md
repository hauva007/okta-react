# okta-react
Okta-React Web App

Web app uses [Okta authentication](https://www.okta.com/)

## Installation

Install `okta-react` with [npm](https://www.npmjs.com/):

```
$ npm install
```

## run

Run `okta-react` with [npm](https://www.npmjs.com/):

```
$ npm start
```

Open [browser](http://localhost:3000/) to use web app
